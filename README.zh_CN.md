## Deepin Graphics Driver Manager

Deepin Graphics Driver Manager(deepin-graphics-driver-manager) 便于用户管理显卡驱动.

## 安装

### 从源代码构建

1. 确保已经安装了所有的编译依赖.
````bash
sudo apt build-dep
````

2. 构建:

```bash
mkdir build
cd build
cmake ..
make
```

3. 安装:

```bash
sudo make install
```
